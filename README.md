## 🧐 Sobre a atividade 

Utilizando o Node.js, crie uma simples calculadora, aplicando os conceitos de módulos. (Não precisa estilizar uma página, os dados devem ser passados via url. Exemplo: /calculador/soma/{valor1}/{valor2}).
